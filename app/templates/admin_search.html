한국 도서 검색 API Key 필요

한국어 도서를 검색하려면 카카오 또는 네이버 API Key가 필요합니다.
아래 방법 중 하나로 설정해주세요.

🟡 카카오 책 검색 (권장, 무료)

  1. developers.kakao.com 가입 → 앱 만들기
  2. 앱 → 앱 키 → REST API 키 복사
  3. .env 파일에 KAKAO_REST_API_KEY=발급받은키 추가

🟢 네이버 책 검색 (무료)

  1. developers.naver.com → 애플리케이션 등록
  2. Client ID / Secret 복사
  3. .env에 NAVER_CLIENT_ID= / NAVER_CLIENT_SECRET= 추가 후 서버 재시작

검색 결과가 없습니다

다른 제목이나 저자명으로 다시 검색해보세요.
